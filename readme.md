## Controls - also on MainMenu
Movement [WASD]

Jump/Double Jump [Space]

Shoot [Hold/Release LMB]

Dash [LShift]

Grenade [F]

Grenade Mode [Q]

(dashing, grenades and releasing the bow must be done in time with the music)
